# Adidas-Sales-Analysis-Performance-Dashboard

Project Overview

This project presents an interactive Adidas Sales Analysis Dashboard designed to analyze sales performance, profit trends, retailer contribution, and geographic distribution. The dashboard provides key business insights through data visualization and KPI tracking.
The objective is to transform raw sales data into actionable insights to support strategic business decisions.

A few major retailers contribute a significant portion of total sales.

Business Value

This dashboard helps stakeholders monitor sales performance, identify top-performing products and sales channels, evaluate retailer effectiveness, and optimize geographic expansion strategies. It supports data-driven decisions for revenue growth and operational efficiency.
